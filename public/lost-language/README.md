# Lost Language (BYOD ver.)

![Thumbnail](./thumbnail.jpg)

## 解説

この世界には、もはや解読できない文字やデータが溢れ、それらの数は増え続けています。デジタルの世界でも同じことが言えます。この作品では、自作のフォントを p5.js に読み込むことで、デジタル空間をさまよう「失われた言語」を表現しました。
